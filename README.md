Download
--------
+ Download latest release from here: https://github.com/Android-RPi3/android_releases_rpi3/releases

How to install
--------------
```
$ tar xvfz android_rpi3_*.tar.gz
$ cd rpi3
$ ./install.sh -p /dev/sdX
```
+ Replace **X** with proper device number.
